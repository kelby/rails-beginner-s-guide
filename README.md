# Rails 参考手册

本书包括两部分：Rails 源码剖析和 Rails 使用指南。Rails 是一个 Web 开发框架，也是一个工具。"工欲善其事必先利其器"，想要更好的使用 Rails 这个工具，清楚其背后的魔法，阅读源代码是必备功课。

### 本书适合什么样的读者？

- 想要更好的使用 Rails
- 准备阅读 Rails 的源代码
- 想知道 Rails 的整体架构
- 想清楚 Rails 背后的魔法

### 本书包含了哪些内容？

- 一些文档里找不到的方法
- 每个模块的关键所在
- 源码阅读路线图

### 本书不讲哪些内容？

- 安装、部署
- 奇技淫巧
- Ruby 语法、Rails 入门
- 和具体业务相关的问题或功能
