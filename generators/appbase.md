## ~~App Base~~

继承于 Base.

**针对 Rails 本身提供了大量方便使用的方法。**注意，它们不具备通用性。更多信息，可以查看源代码，这里就不列出了。

```
rails new APP_PATH [options]
```

**创建新 Rails 应用**的时候，它会派上大用场。

```
rails plugin new APP_PATH [options]
```

**创建新 engine**的时候，也会派上大用场。
