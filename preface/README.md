# Rails 之道

《Rails 之道》是书名，包括两部分 Rails 使用指南，Rails 源码剖析，这是本书的写作目标。Rails 是一个 Web 开发框架，也是一个工具。"工欲善其事必先利其器"，想要更好的使用 Rails，清楚背后的魔法，阅读源代码是必备功课。

## 本书适合什么样的读者？

- 想清楚 Rails 背后的魔法
- 想阅读 Rails 的源代码
- 想知道 Rails 的整体架构

## 本书不讲哪些内容
